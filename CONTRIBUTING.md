# Contributing

Thank you for your interest in FENITH.

This repository serves as the meta-repository for an ongoing PhD research project on Federated Learning applied to European Digital Healthcare.

## How to Contribute

1. **Bug Reports & Issues** — Open an issue describing the problem. Please specify:
   - Which repository or submodule is affected
   - Which Research Question (RQ1–RQ4) is involved, if applicable
   - Steps to reproduce the issue
2. **Suggestions** — Feature requests, research ideas, or structural improvements are welcome via GitHub Issues.
3. **Pull Requests** — For documentation fixes, typos, or structural improvements to the meta-repository.

## Submodule Contributions

Each submodule has its own repository and may have its own contribution guidelines. Please refer to the specific repository for details:

- For **platform** contributions (flopbg, BLEKFL2, FL-EHDS-FLICS2026), open issues/PRs in the respective repository.
- For **paper** repositories, contributions are generally not accepted as they reflect published or submitted academic work.

## How to Cite

If you use or reference this work, please cite the foundational paper using the metadata in [`CITATION.cff`](CITATION.cff):

> Liberti, F.; Berardi, D.; Martini, B. *Federated Learning in Dynamic and Heterogeneous Environments: Advantages, Performances, and Privacy Problems.* Applied Sciences 2024, 14(18), 8490. [DOI: 10.3390/app14188490](https://doi.org/10.3390/app14188490)

## Code of Conduct

All interactions should follow standard academic courtesy and professional conduct.
