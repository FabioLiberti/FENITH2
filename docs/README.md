# Documentation

This directory contains all research documentation for the FENITH meta-repository.

## Structure

| Directory | Contents |
|---|---|
| [`thesis/`](thesis/) | Thesis outline, extended abstract, dataset synthesis, and scientific production evaluation |
| [`papers/`](papers/) | Published and submitted paper PDFs — see [`INDEX.md`](papers/INDEX.md) for the full mapping |
| [`questionnaire/`](questionnaire/) | Survey materials for the FL adoption questionnaire in Italian hospitals |
| [`architecture.md`](architecture.md) | Mermaid diagram of the FENITH research architecture |

## Papers Index

The [`papers/INDEX.md`](papers/INDEX.md) file maps each publication code (P-M, P-IS, P-CR, etc.) to its local PDF filename, title, venue, year, status, and DOI.
